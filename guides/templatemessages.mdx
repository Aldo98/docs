# Template Messages

Template messages are predefined, highly structured notifications (HSM). After your account is fully set up, you can create these templates and submit them to WhatsApp for review and approval. Once approved, you can use them to message users who have provided their consent (opted-in). Within the Harkha, all your Message Templates can be organized and managed centrally through the

## Create New Template

To create a new Template, navigate to Harkhas Dashboard \> Template \> Create Template. Fill in the required details for the message you want to create. The descriptions and instructions for each field are provided below.

| Parameter        | Definition                                                                                                            | Restrictions                                     | Required |
| ---------------- | --------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------ | -------- |
| Template Name    | Defines the unique name of the template. Allowed characters include lowercase letters, numbers, and underscores only. | Spaces and special characters are not permitted. | Yes      |
| Category         | Select the category that best represents the purpose of your template.                                                | —                                                | Yes      |
| Template Label   | Specify the purpose of the template in 2–3 words (e.g., Account Update, OTP).                                         | —                                                | Yes      |
| Header Type      | Defines the header of your message template. Supported types include Image, Video, Document, Text, and Location.      | Maximum 60 characters (for text headers).        | No       |
| Body Text        | Contains the main content of your message template.                                                                   | Maximum 1,024 characters.                        | Yes      |
| Add Variable     | Allows the use of parameter placeholders in the format `{{1}}`.                                                       | —                                                | Yes      |
| Footer Text      | Optional footer text displayed at the bottom of the message template.                                                 | Maximum 60 characters.                           | No       |
| Add URL          | Adds a clickable URL button to the template.                                                                          | —                                                | No       |
| Add Phone Number | Adds a call button with a phone number to the template.                                                               | —                                                | No       |
| Quick Reply      | Enables quick reply buttons for user interaction.                                                                     | —                                                | No       |

## Your Template

This page displays all your WhatsApp (Meta)–approved templates. It includes a search bar and multiple filter options to help you quickly locate the template you need. Templates can be filtered by the industry selected during creation, by template topic, or by their intended use case.

## Template Library

This page allows you to configure sample messages for any category you require.
You can filter templates by industry, ensuring the content aligns with common messaging standards within that sector. You may also filter by template topic to quickly locate messages related to a specific subject. Finally, the use-case filter helps you narrow down templates based on the most relevant business scenarios.
