# Webhook

Webhooks provide a way for your application to receive real-time updates whenever specific events occur within your account. They function as HTTP callbacks that notify your system instantly, enabling your application to respond quickly and efficiently to relevant events.

![webhook](images/webhook.png)

Within this section, you can enable or disable the webhook using the provided toggle. Additionally, you may select the event types—Message Events and System Events—through checkboxes to specify which notifications should be delivered to your webhook endpoint.
